Journals by Discipline Tool

This tool provides a view of the usage of our collection in a specified Discipline. Select a Discipline and then view metrics available for each title assigned to that Discipline. All journals by Discipline which have at least one (1) of the chosen metric (downloads, publications, citations) are displayed. Journals which have zero (0) use for a given metric are not displayed. To see all journals by metric within a discipline (including those with zero usage), select the “Show Data in Table” view option, just below the Disciplines dropdown menu.
